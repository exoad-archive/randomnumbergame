## randomnumbergame
A random Number Game that is created using the Java language.

# Usage:
**How to run:**
Basic for most Java files.
If not:
javac RandNum.java
java RandNum.java

--{---}--

Theres 3 levels:
Easy
Medium
Hard
Impossible
Custom

**Workings of custom:**
Custom is still a work in progress

The Game uses the `java.util.Random` Package
